Títulos Liquidados no Dia

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo listar títulos liquidados no dia para o cedente selecionado.
 
Botões de Ação:
Ok: Encaminha para funcionalidade de consulta do título liquidado no dia.
Voltar: Retorna para tela anterior.
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física do cedente.
Tipo de Pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
Código Cliente (COCLI): Código do cliente no SICLI.
Data Crédito: Data de crédito dos valores na conta do cedente.
Nosso Número: Código identificador do título no banco.
Valor Pagamento: Valor de liquidação dos títulos.
Meio Liquidação: Canal utilizado para liquidação do título.
Forma: Forma utilizada para pagamento do título.
