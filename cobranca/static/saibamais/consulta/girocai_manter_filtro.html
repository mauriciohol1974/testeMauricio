GiroCaixa Instantâneo - Filtro

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo filtrar para consulta do GiroCaixa.
Esta funcionalidade é somente para Pessoa Jurídica.
 
Botões de Ação:
Ok: Encaminha para funcionalidade de listagem selecionada.
 
Campos:
Cedente: Código do cedente.
CNPJ: Tipo de pessoa e ao lado o número do CNPJ(Pessoa Jurídica).
Campo de tipo de pessoa desabilitado para edição.
Unidade: Seleciona a Unidade e ao lado o código da Unidade.
