Títulos Liquidados - Consultar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo consultar os títulos liquidados.
 
Botões de Ação:
Voltar: Retorna para funcionalidade de listagem dos títulos liquidados do cedente .
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física do cedente.
Tipo de Pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
Código Cliente (COCLI): Código do cliente no SICLI.
Data Pagamento: Data de pagamento do título.
Número Docto: Código identificador do título para o cedente.
Valor Título: Valor nominal do título.
Nosso Número: Código identificador do título no banco.
Situação: Situação do título.
Data Situação: Data da atual situação do título.
Data Vencimento: Data de vencimento do título.
Modalidade: Código da modalidade do título.
Data Crédito: Data de crédito do cedente.
Meio de Entrada: Canal utilizado como meio de entrada do título no sistema do banco, se registrado.
Moeda: Tipo de moeda do título.
PV Recebedor: Código do PV recebedor do título.
PV Cobrador: Código do PV cobrador do título.
Espécie: Espécie do título de cobrança.
Aceite: Tipo de aceite.
Número Lote: Código identificador do número de lote quando ocorrer liquidação do título via Compensação.
Quantidade Moeda: Quantidade da moeda.
Abatimento: Valor de abatimento do título.
Data Entrada: Data do registro do título no sistema do banco.
Valor Juros: Valor total de juros do título.
Percentual de Juros: Percentual de juros do título.
Valor Multa: Valor total de multa do título.
Percentual de Multa: Percentual de multa do título.
Data Multa: Data para começo de cobrança da multa.
Prazo Multa: Prazo em dias para cobrança da multa no título.
Valor Desconto 1, 2 e 3: Valor de desconto para o título.
Desconto Percentual 1, 2 e 3: Percentual de desconto para o título.
Data Desconto 1, 2 e 3: Data para começo de cobrança do desconto no título.
Prazo Desconto 1, 2 e 3: Quantidade de dias para vigência do desconto.
Data Emissão: Data de emissão do título liquidado.
Data para Protesto: Data prevista para encaminhar título ao cartório.
Prazo Protesto: Prazo em dias para o protesto.
Valor Cobrado: Valor de liquidação do título.
