Histórico de Cedentes - Consultar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo consultar histórico do cedente.
 
Botões de Ação:
Voltar: Retorna para tela anterior.
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física do cedente.
Tipo de pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
Código Cliente (COCLI): Código do cliente no SICLI.
Filtro Guia: Guia selecionada para filtragem.
Filtro Data de: Data inicial da filtragem.
Filtro Data até: Data final da filtragem.
Informação Alterada: Informação alterada no cadastro do cedente.
Data Movimento: Data da alteração no cadastro do cedente.
Hora: Hora da alteração no cadastro do cedente.
Usuário: Código do Usuário de alçada responsável pela alteração.
Valor Anterior: Valor antes cadastrado.
Valor Atual: Valor alterado pelo Usuário.
