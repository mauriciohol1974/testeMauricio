Títulos Alterados - Consultar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo consultar os títulos alterados do PV informado.
 
Botões de Ação:
Ações: Encaminha o título selecionado para funcionalidade de ações do banco de títulos.
Voltar: Retorna para funcionalidade de filtragem dos títulos alterados por Unidade PV.
 
Campos:
Unidade PV: Código do PV solicitado.
Nome Unidade: Nome do PV solicitado.
Data Comando: Data da filtragem.
Comando: Tipo de alteração no título.
Data Alteração: Data da alteração do título.
Nosso Número: Código identificador do título no banco.
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física do cedente.
