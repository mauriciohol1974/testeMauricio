Solicitações no Dia -
Emissão de Título para Banco de Sacados - Listar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo listar banco de sacados com emissão de títulos.
 
Botões de Ação:
Ok: Encaminha para funcionalidade de consulta de emissão do título do banco de sacados.
Voltar: Retorna para funcionalidade de filtragem dos serviços solicitados no dia.
 
Campos:
Cedente: Código do cedente.
Nome Fantasia: Nome da Pessoa Jurídica ou da Pessoa Física do cedente.
Tipo de Pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
Código Cliente (COCLI): Código do cliente no SICLI.
Relatório: Nome do item selecionado na filtragem.
Número do Banco: Código do banco de sacados.
Nome do Banco: Nome do banco de sacados.
