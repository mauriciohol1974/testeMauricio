GiroCaixa Instantâneo (Cedente) - Consultar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo consultar cedente com GiroCaixa .
 
Botões de Ação:
Voltar: Retorna para tela anterior.
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica do cedente.
Tipo de pessoa: Tipo de pessoa do cedente.
CNPJ: Número do CNPJ(Pessoa Jurídica) do cedente.
Código Cliente (COCLI): Código do cliente no SICLI.
Atividade Econômica: Código da atividade econômica do cedente.
Data da Inclusão: Data de inclusão do cedente.
GiroCaixa Escritural: Se GiroCaixa Escritural ou não.
Data da Última Alteração: Data da última alteração do cedente.
Matrícula do Concessor: Código de matrícula do concessor de GiroCaixa.
Data da Exclusão: Data de exclusão do cedente.
Unidade PV: Código da Unidade PV.
Nome Unidade PV: Nome da Unidade PV.
Unidade EN: Código da Unidade EN.
Nome Unidade EN: Nome da Unidade EN.
