Consultas Gerenciais - Listar

Descrição Funcionalidade:
Esta funcionalidade apresenta a listagem de acordo com o filtro selecionado.
Estas são funcionalidades não consolidadas.
 
Listar Caixa:
 
Botões de Ação:
Ok: Encaminha para funcionalidade de listagem das Unidades PV para a Unidade EN selecionada.
Voltar: Retorna para funcionalidade de filtragem.
 
Campos:
Tipo de Data: Tipo da data do filtro.
Data: Data atual do sistema.
Tipo de Consulta: Tipo de consulta selecionado.
Relatório: Tipo de relatório.
Unidade EN: Código da Unidade EN.
Nome Unidade: Endereço da Unidade EN.
 
Listar Unidade EN:
 
Botões de Ação:
Ok: Encaminha para funcionalidade de listagem dos cedentes para a Unidade PV selecionada.
Voltar: Retorna para a tela anterior.
 
Campos:
Tipo de Data: Tipo da data do filtro.
Data: Data atual do sistema.
Tipo de Consulta: Tipo de consulta selecionado.
Relatório: Tipo de relatório.
Unidade EN: Código da Unidade EN.
Nome Unidade: Nome da Unidade EN.
Unidade PV: Código da Unidade PV.
Nome Unidade: Endereço da Unidade PV.
 
Listar Unidade PV:
 
Botões de Ação:
Ok: Encaminha o para a funcionalidade de listagem dos cedentes para a Unidade PV selecionada.
Voltar: Retorna para a tela anterior.
 
Campos:
Tipo de Data: Tipo da data de pesquisa.
Data: Data atual do sistema.
Tipo de Consulta: Tipo de consulta selecionado.
Relatório: Tipo de relatório.
Unidade PV: Código da Unidade PV.
Nome Unidade: Endereço da Unidade PV.
Cedente: Código do redente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física.
Tipo Pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
