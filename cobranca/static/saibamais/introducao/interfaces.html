1.3. Interfaces do Sistema

O SIGCB possui interfaces com:

 
SICOB: Gerar arquivo com os boletos recebidos.
SIDEC: Envia todas as informações para efetivação de débito/crédito nas Contas Correntes dos Cedentes em D=0, no entanto os float negociados para cada canal serão respeitados.
SINAF: O sistema de cobrança sensibilizará os eventos contábeis específicos por meio do SINAF.
SICLI: Sistema integrado on line para consulta a dados cadastrais do cedente.
SIBNC: Recebe arquivo referente a dados de negócio constantes na base de dados do sistema.
SIDEM: Recebe arquivo contendo dados gerenciais.
SIAPV: Recebe boletos de pagamento, gera interface online para validação de dados. Recebe arquivo com tabela de Cedentes Ativos.
SIAUT: Recebe boletos de pagamento, gera interface online para validação de dados. Recebe arquivo com tabela de Cedentes Ativos
SITAE: Recebe boletos de pagamento, gera interface online para validação de dados. Recebe arquivo com tabela de Cedentes Ativos
SIICO: Disponibiliza tabela de unidades e vinculações (através de subrotinas ICOBBnnn).
SIRUC: Disponibiliza informações sobre rotinas de uso comum para cálculo de datas, dígitos verificadores, tratamento de relatórios, obtenção de informações de segurança, etc.
