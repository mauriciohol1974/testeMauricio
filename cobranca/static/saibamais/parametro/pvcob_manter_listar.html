Manter PV Cobrador - Listar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo listar por Unidade PV Cobrador.
 
Botões de Ação:
Alterar: Encaminha para funcionalidade de alterar a Unidade PV Cobrador selecionada.
Excluir: Alterna a situação da Unidade PV Cobrador para "Inativa" (Exclusão lógica).
Esta Unidade só será removida do banco de dados após uma rotina batch de expurgo.
Voltar: Retorna para funcionalidade de filtrar Unidade PV Cobrador para manutenção.
 
Campos:
Unidade PV De: Código da Unidade PV Cobrador inicial.
Até: Código da Unidade PV Cobrador final.
Unidade PV: Código da Unidade PV Cobrador.
Apelido: Apelido da Unidade PV Cobrador.
Nome Unidade: Nome da Unidade PV Cobrador.
Padrão: Tipo do arquivo padrão da Unidade PV Cobrador.
Num. Remessa: Número da remessa.
Qtd. Reg.: Quantidade de registros.
Data Remessa: Data da remessa.
Situação: Situação da Unidade PV Cobrador.
