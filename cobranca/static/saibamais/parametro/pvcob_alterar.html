Manter PV Cobrador - Alterar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo alterar a Unidade PV Cobrador.
 
Botões de Ação:
Confirmar: Confirma a alteração da Unidade PV Cobrador.
Voltar: Retorna para funcionalidade listar Unidade PV Cobrador.
 
Campos:
Unidade PV: Código da Unidade PV Cobrador.
Nome Unidade: Nome da Unidade PV Cobrador.
Endereço: Endereço da Unidade PV Cobrador.
E-mail: E-mail da Unidade PV Cobrador.
 
Guia Faixas de CEP
Botões de Ação:
Mostrar: Mostra o número de faixas de CEP selecionado.
 
Campos:
Faixa de CEP de: Número do CEP inicial.
Faixa de CEP até: Número do CEP final.
No. de Faixas de CEP: Selecionar o número de faixas de CEP.
 
Guia Dados Complementares
Campos:
VAN: Tipo de VAN.
Desabilitado para alteração.
Automação Cartorária: Selecionar se tem automação cartorária ou não.
Se automação cartorária for "não", os campos 'Apelido VAN' e 'Arquivo Padrão' serão desabilitados.
Desabilitado para alteração.
Apelido VAN: Apelido para a VAN.
Desabilitado para alteração.
Arquivo Padrão: Selecionar o tipo de arquivo padrão.
Desabilitado para alteração.
Número Última Remessa: Número da última remessa enviada.
Quantidade de Registros: Quantidade de registros enviados.
Data Última Remessa: Data da última remessa enviada.
Inclusão na Automação: Data da inclusão na automação cartorária.
Usuário Responsável: Usuário de alçada responsável pela alteração.
Data Última Alteração: Data da última alteração.
Situação: Selecionar a situação da Unidade.
Se o campo 'Situação' for "Ativa",estará desabilitado.
