2.5. Menu: Menu Principal do Sistema

Descrição Funcionalidade:
O menu de acesso as funcionalidades será montado de acordo com o perfil do Usuário. Para navegar no sistema, coloca-se o “mouse” sobre o nome de um dos módulos disponíveis no menu, e na sequência será aberta uma caixa de opções de funcionalidades do módulo ou um sub-menu. Para escolher uma das funcionalidades ou sub-menu do módulo, clique com o botão esquerdo do “mouse” sobre a opção desejada.

Os módulos disponíveis no Menu são:
  • Cedentes
  • Serviços
  • Consultas
  • Parâmetros
  • Saiba Mais
  • Sair
Para detalhes sobre cada módulo e suas funcionalidades acesse a seção funcionalidades na tela inicial do sistema.
