2.1. Padrões de Navegação

Para acesso as funcionalidades do sistema, será utilizado o seguinte padrão de navegação:
  1. Realizar login no sistema;
  2. Acessar a tela de menu no sistema;
  3. Selecionar uma das funcionalidades que aparecem no menu;
  4. A tela da funcionalidade será exibida;
  5. No processamento da funcionalidade, podem ser exibidas mensagens do sistema ao usuário;
  6. Caso ocorram erros no processamento, será exibida tela padronizada de erro.
  7. Dúvidas sobre a funcionalidade podem ser esclarecidas na funcionalidade saiba mais.
  8. Para sair do sistema é indicado executar a funcionalidade sair.
