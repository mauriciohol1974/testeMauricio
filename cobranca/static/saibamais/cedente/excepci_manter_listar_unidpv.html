Excepcionação de Pendência de Alçada - Listar PV

Descrição Funcionalidade:
Esta funcionalidade é listar cedentes com excepcionação, por Unidade PV.
 
Botões de Ação:
Ok: Encaminha o usuário para a funcionalidade de ações do cedente selecionado.
Voltar: Retorna para a tela anterior.
 
Campos:
Unidade PV: Código do PV.
Nome Unidade: Nome do PV.
Cedente: Código do Cedente.
Nome Cedente: Nome fantasia para Pessoa Jurídica ou nome para Pessoa Física.
Tipo de Pessoa: Tipo de pessoa do cliente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica).
