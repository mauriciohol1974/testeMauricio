Manter Cedente - Consulta Detalhada

Descrição Funcionalidade:
Esta funcionalidade é consultar o dados de cadastro do cedente.
 
Botões de Ação:
Alterar: Encaminha para funcionalidade de alterar o cedente.
Exclui: Exclui o cedente.
Voltar: Retorna para funcionalidade de filtrar o cedente para manutenção.
 
Campos:
Código Cliente (COCLI): Código do cliente no SICLI.
Carteira: Código identificador da carteira.
Tipo de Pessoa: Tipo de pessoa do cliente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica).
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física.
Razão Social: Nome completo da Pessoa Jurídica. Em caso de Pessoa Física este campo ficará em branco.
e-mail: E-mail do cliente.
Endereço: Endereço do cliente.
Cedente: Código do cedente.
 
Guia Geral
Descrição da Guia:
Nesta guia serão consultados os dados para identificação do cedente.
 
Campos:
Tipo de Cobrança: Tipo de cobrança.
Se for "Convencional", a 'Guia Cedente Eletrônico' ficará como "Não Permitido", não podendo ser consultada.
Se for "Convencional", o campo 'Cedente Centralizador' ficará vazio.
Protesto Automático: Se protesto automático ou não.
Prazo Protesto: Prazo em dias para envio do título vencido ao cartório.
Prazo Devolução: Prazo em dias para devolução do título após vencimento.
Extrato Mov. Títulos: Se tem extrato de movimentação de títulos ou não.
Se este campo for "não", o campo 'Destino Extrato Mov.' ficará vazio.
Destino Extrato Mov.: Modo de envio do extrato de movimentação de títulos.
Extrato Mov. Déb./Créd.: Se tem extrato de movimentação para débitos e créditos ou não.
Se este campo for "não", o campo 'Destino Extrato Déb./Créd.' ficará vazio.
Este campo só poderá ser "sim" se o campo 'Extrato Mov. Títulos' for "sim".
Destino Extrato Déb./Créd.: Modo de envio do extrato de movimentação para débitos e créditos.
Inventário Mês: Se tem inventário do mês ou não.
Tipo de Juros Dia: Tipo de juros que será utilizado em caso de título pago após o vencimento, se for o caso.
Se o tipo de juros for "Juros Caixa" ou "Sem Juros", o campo 'Percentual Juros Dia' ficará vazio.
Percentual Juros Dia: Percentual de juros por dia cobrado sobre o valor do título após o vencimento.
Recebimento em Cheque: Se aceita recebimento em cheque ou não.
Percentual Multa: Percentual da multa cobrado sobre o valor do título após o vencimento.
Prazo para Multa: Prazo em dias para cobrança da multa.
Banco de Sacados: Se possui banco de sacados ou não.
Cliente Conta Garantida: Se o cliente vai Descontar Títulos ou não.
Cliente Externo: Se é cliente externo ou não.
Campo vazio para Pessoa Física.
Cliente Giro Caixa: Se utiliza GiroCaixa ou não.
Campo vazio para Pessoa Física.
Se for "Convencional" ou "Eletrônica", o campo 'Setor' terá apenas a opção "Privado Pessoa Jurídica".
Setor: Setor de atuação da Pessoa Jurídica.
Campo vazio para Pessoa Física.
Porte: Porte da empresa da Pessoa Jurídica.
Campo vazio para Pessoa Física.
Natureza: Natureza do cedente.
Campo vazio para Pessoa Física.
Ramo de Atividade: Ramo de atividade da Pessoa Jurídica.
Campo vazio para Pessoa Física.
Exclusão Automática: Se exclusão automática do cedente da base, após dias sem movimentação.
Modalidade de Título: Modalidade do título de cobrança.
PV de Vinculação: Código da Unidade PV de vinculação do cliente, onde foi contratada a cobrança.
Cedente Centralizador: Preenchido se o cedente for eletrônico.
Banco Correspondente: Código do banco correspondente.
Cobrança Sem Boleto: Se cobrança é sem boleto ou não.
Situação do Cedente: Indicador da situação do cedente.
 
Guia Float
Descrição da Guia:
Nesta guia serão consultadas as quantidades de dias para crédito na conta do cedente.
 
Campos:
Padrão: Quantidade de dias padrão.
Negociado: Quantidade de dias negociado com o cliente.
 
Guia Contas
Descrição da Guia:
Nesta guia serão consultadas as contas do cedente para crédito, débito e rateio.
 
Campos:
Conta Crédito: Código da Unidade, operação, número e dígito da conta para crédito do cedente e esta será a conta para crédito principal.
Perc. Participação: Percentual de participação para a conta de crédito principal.
Conta Débito: Código da Unidade, operação, número e dígito da conta principal para débito das tarifas e serviços contratados pelo cedente.
Perc. Participação: Percentual de participação para a conta de débito principal.
Conta Caução: Código da Unidade, operação, número e dígito da conta caução do cedente. Esta conta só será utilizada caso o cedente contrate a operação Caução de Títulos.
Contas de rateio de valores: Código da Unidade, operação, número e dígito da conta para rateio de valores.
Perc. Crédito: Percentual para rateio do crédito.
Valor Crédito: Valor do crédito a ser rateado.
Se este campo estiver preenchido, o campo 'Perc. Participação' para o crédito ficará em branco.
Perc. Débito: Percentual para rateio do débito.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica).
Titular: Nome do titular da conta para rateio cadastrada.
 
Guia Cedente Eletrônico
Descrição da Guia:
Nesta guia será consultado o modo de comunicação entre a CAIXA Econômica Federal e o cedente.
 
Campos:
Aplicativo: Aplicativo utilizado na transmissão.
Tipo de Transmissão: Tipo de transmissão.
Se for "Linha Dedicada" ou "Internet", o campo 'VAN' ficará vazio.
Padrão de Arquivo: Padrão de arquivo de transmissão.
Agrupamento: Agrupamento de serviços.
Atribuição de VAN: Quem faz atribuição de VAN.
VAN: VAN de comunicação.
Junção Arquivo de Retorno: Se tem junção de arquivo de retorno ou não.
Se for "não", o campo "Cedente Junção" ficará vazio.
Cedente Junção: Código do cedente solicitante da junção de arquivo.
Perfil Rejeição: Perfil de rejeição.
Pré-Crítica: Se haverá pré-crítica ou não.
Apelido: Apelido para o arquivo retorno.
Arquivo de Retorno: Nome do arquivo de retorno.
Cópia Arquivo Retorno: Se haverá cópia do arquivo de retorno.
Se for "não", o campo 'Apelido (Cópia)' ficará vazio.
Caixa Postal: Número da caixa postal do cedente.
Apelido (Cópia): Apelido para a cópia do arquivo de retorno.
Arquivo de Retorno (Cópia): Nome para a cópia do arquivo de retorno.
Número Próxima Remessa: Número da próxima remessa de arquivo a ser enviada pelo cedente.
Número Último Retorno: Número do último arquivo de retorno gerado pelo sistema para o cedente.
 
Guia Boleto
Descrição da Guia:
Nesta guia serão consultadas as informações para o boleto.
 
Campos:
Emissão de Boleto: Emissor do Boleto.
Envio do Boleto: Modo de envio do boleto.
Se for "E-Mail", o campo 'Aviso Disponib. Boleto' estará preenchido.
Aviso ao Sacado: Se envia aviso ao sacado ou não.
Se for "não", os campos 'Envio do Aviso ao Sacado', 'Formas de Aviso', 'Valor Mínimo para Aviso' e 'Aviso Vencido' ficarão vazios.
Envio do Aviso ao Sacado: Modo de envio do aviso ao sacado.
Formas de Aviso: Seleciona o(s) aviso(s) que deseja enviar.
Se for "Vencido", o campo 'Aviso Vencido' estará preenchido.
Valor Mínimo para Aviso: Valor a ser considerado para emissão do aviso, faz referência ao valor nominal do título.
Quantidade de Emissões: Número de emissão do boleto.
Aviso Disponib. Boleto: Quantidade de dias para aviso de disponibilidade de boleto.
Aviso Vencido: Quantidade de dias para aviso de boleto vencido.
 
Guia Mensagens para Boletos
Descrição da Guia:
Nesta guia serão consultadas as mensagens para serem inseridas nos boletos, se emitidos pela CAIXA.
 
Campos:
Boleto Padrão:
Local de Impressão da Mensagem: Qual(is) local(is) saem a(s) impressão(ões).
Mensagem Recibo do Sacado: Mensagem para ser impressa no Recibo do Sacado.
Mensagem Verso Boleto: Mensagem para ser impressa no verso do boleto.
Mensagem Ficha Compensação: Mensagem para ser impressa na Ficha de Compensação.
 
Boleto Personalizado:
Local de Impressão da Mensagem: Qual(is) local(is) saem a(s) impressão(ões).
Mensagem Recibo do Sacado: Mensagem para ser impressa no Recibo do Sacado.
Mensagem Verso Boleto: Mensagem para ser impressa no verso do boleto.
Mensagem Ficha Compensação: Mensagem para ser impressa na Ficha de Compensação.
Mensagem Recibo do Sacado A: Mensagem para ser impressa no Recibo do Sacado A (Modelos 10 e 11).
Mensagem Recibo do Sacado B: Mensagem impressa no Recibo do Sacado B (Modelo 11).
 
Boleto PRé-Impresso:
Local de Impressão da Mensagem: Seleciona qual(is) local(is) devem sair a(s) impressão(ões).
Mensagem Recibo do Sacado (A4): Mensagem para ser impressa no Recibo do Sacado.
Mensagem Ficha Compensação (A4/Matricial): Mensagem para ser impressa na Ficha de Compensação do modelo matricial ou A4.
 
Banco Correspondente:
Mensagem: Mensagem para ser impressa no Boleto.
 
Guia Tarifas
Descrição da Guia:
Nesta guia serão consultadas as tarifas para o(s) boleto(s).
 
Botão de Ação:
Verificar: Consulta de tarifas para o grupo selecionado.
 
Campos:
Periodicidade da Tarifa: Como deverão ser efetuados os débitos na conta do cedente.
Somente se for "mensal", o campo 'Dia do Débito' estará preenchido.
Dia do Débito: Dia para débito da(s) tarifa(s).
Forma de Cálculo: Forma de cálculo da tarifa.
Grupo: Seleciona o grupo de tarifas que deseja consultar.
 
Verificar Tarifas:
Campos:
Serviço: Nome do serviço no grupo que será cobrada a tarifa.
Valor Orig.: Valor de tarifa padrão.
Perc. Orig.: Percentual de tarifa padrão.
Valor Calc.: Valor de tarifa negociado.
Perc. Calc.: Percentual de tarifa negociado.
 
Guia Conclusão
Descrição da Guia:
Nesta guia são consultados dados de conclusão da operação de inclusão e alteração do cedente.
 
Campos:
EN: Código do EN de cadastro do cedente.
Data de Inclusão: Data de cadastro do cedente.
Data Última Alteração: Data da última alteração do cedente.
Responsável Alteração: Código do Usuário responsável pela alteração.
