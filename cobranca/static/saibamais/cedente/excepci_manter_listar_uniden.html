Excepcionação de Pendência de Alçada - Listar EN

Descrição Funcionalidade:
Esta funcionalidade é listar as Unidades PV por vinculação à Unidade EN.
 
Botões de Ação:
Ok: Encaminha para funcionalidade de listagem dos cedentes vinculados por Unidade PV.
Voltar: Retorna para tela anterior.
 
Campos:
Unidade EN: Código do EN.
Nome Unidade: Nome do EN.
Unidade PV: Código do PV.
Nome Unidade: Endereço do PV.
