Manter Cedente (PV) - Listar

Descrição Funcionalidade:
Esta funcionalidade é listar cedentes por Unidade PV.
 
Botões de Ação:
Ações: Encaminha o usuário para funcionalidade de ações do cedente.
Alterar: Encaminha o usuário para funcionalidade de alterar cedente.
Excluir: Alterna a situação do Cedente para "Inativo" (Exclusão lógica).
Este cedente só será removido do banco de dados após rotina batch de expurgo.
Consulta: Encaminha o usuário para a tela de consulta do Cedente.
Voltar: Retorna para funcionalidade de filtrar para manutenção do cedente.
 
Campos:
Unidade PV: Código da Unidade PV.
Nome Unidade: Nome da Unidade PV.
Cedente: Código do Cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica).
Nome Cedente: Nome fantasia para Pessoa Jurídica ou nome para Pessoa Física.
COCLI: Código do cliente no SICLI.
Situação: Situação do Cedente.
