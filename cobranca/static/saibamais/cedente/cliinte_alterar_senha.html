Manter Senha Cliente Internet - Alterar

Descrição Funcionalidade:
Esta funcionalidade é a de alteração da senha do Cliente Internet.
 
Botões de Ação:
Confirmar: Conclui a alteração de senha do usuário.
Voltar: Retorna para tela anterior.
 
Campos:
Cedente: Código do Cedente.
Nome Cedente: Nome fantasia para Pessoa Jurídica ou nome para Pessoa Física.
Tipo de Pessoa: Identificador do tipo de pessoa.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica).
Código Cliente (COCLI): Código do cliente no SICLI.
CPF do Usuário: Número do CPF do usuário.
Nome: Nome do usuário.
Senha: Nova senha do usuário Cliente Internet.
Repetir Senha: Redigitar a nova senha do usuário Clente Internet.
