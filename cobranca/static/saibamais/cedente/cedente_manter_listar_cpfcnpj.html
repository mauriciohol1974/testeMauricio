Manter Cedente (CPF/CNPJ) - Listar

Descrição Funcionalidade:
Esta funcionalidade é listar cedentes por CPF(Pessoa Física) ou CNPJ (Pessoa Jurídica).
 
Botões de Ação:
Ações: Encaminha para funcionalidade de ações do cedente.
Alterar: Encaminha para funcionalidade de alterar cedente.
Excluir: Alterna a situação da Unidade PV Cobrador para "Inativa" (Exclusão lógica).
Esta Unidade só será removida do banco de dados após rotina batch de expurgo.
Consulta: Encaminha para funcionalidade de consultar cedente.
Voltar: Retorna para funcionalidade de ações do cedente.
 
Campos:
Tipo Pessoa: Tipo de pessoa do cedente (Física ou Jurídica).
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física.
COCLI: Código do cliente no SICLI.
Situação: Situação do cedente.
