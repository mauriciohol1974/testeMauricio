Manter Cedente - Ações

Descrição Funcionalidade:
Esta funcionalidade é de ações para manutenção do cedente.
 
Botões de Ação:
Voltar: Retorna para funcionalidade de filtrar o cedente para manutenção.
 
Campos:
Código do Cliente (COCLI): Código do cliente no SICLI.
Carteira: Código identificador da carteira.
Tipo de Pessoa: Tipo de pessoa do cliente (Física ou Jurídica).
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica).
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física.
Razão Social: Nome completo da Pessoa Jurídica. Em caso de Pessoa Física este campo ficará em branco.
e-mail: E-mail do cliente.
Endereço: Endereço do cliente.
Cedente: Código do cedente.
Espécie Título: Espécie do título de cobrança.
Qtd Vencidos: Quantidade de títulos vencidos.
Valor Vencidos: Valor de títulos vencidos.
Qtd Vencendo no dia: Quantidade de títulos vencendo no dia.
Valor Vencendo no dia: Valor de títulos vencendo no dia.
Qtd a Vencer: Quantidade de títulos a vencer.
Valor a Vencer: Valor de títulos a vencer.
Qtd Liquidados no dia: Quantidade de títulos liquidados no dia.
Valor Liquidado no dia: Valor de títulos liquidados no dia.
 
Ações:
Banco de Títulos: Encaminha para a funcionalidade de filtrar banco de títulos.
Consultas Gerencias: Encaminha para a funcionalidade de filtrar consultas gerenciais.
Consultar Dados do Cedente: Encaminha para a funcionalidade de consultar cedente para manutenção.
Incluir Cedente: Encaminha para a funcionalidade de incluir cedente.
Manter Dados do Cedente: Encaminha para a funcionalidade de alterar cedente para manutenção.
Consultar Rentabilidade do Cedente: Encaminha para a funcionalidade de rentabilidade do cliente para consultas gerais.
Consultar Movimento Mensal (Totais): Encaminha para a funcionalidade de movimento mensal por totais para consultas gerais.
Consultar Movimento Mensal (Extrato): Encaminha para a funcionalidade de movimento mensal por extrato para consultas gerais.
