Manter Cliente Internet (CPF/CNPJ) - Listar

Descrição Funcionalidade:
Esta funcionalidade é listar cedentes por CPF(Pessoa Física) ou CNPJ (Pessoa Jurídica).
 
Botões de Ação:
Alterar Usuário: Encaminha o usuário para a funcionalidade de alteração do usuário.
Alterar Relacionamento: Encaminha o usuário para a funcionalidade de alteração do relacionamento.
Alterar Senha: Encaminha o usuário para a funcionalidade de alteração da senha.
Excluir Relacionamento: Exclui o cliente internet e o seu relacionamento.
Voltar: Retorna para funcionalidade de filtrar.
 
Campos:
Tipo de Pessoa: Tipo de pessoa.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica).
Cedente: Código do Cedente.
Nome Cedente: Nome fantasia para Pessoa Jurídica ou nome para Pessoa Física.
CPF Usuário: Número do CPF do usuário.
Nome Usuário: Nome do usuário.
