Excepcionação de Pendência de Alçada - Alterar Data

Descrição da Guia:
Esta é a funcionalidade de alterar a data de vigência da pendência.
 
Botões de Ação:
Confirmar: Conclui a alteração da data de vigência.
Voltar: Retorna para tela anterior.
 
Campos:
Cedente: Código do Cedente.
Nome Cedente: Nome fantasia para Pessoa Jurídica ou nome para Pessoa Física.
Tipo Pessoa: Tipo de pessoa do Cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica).
Código Cliente (COCLI): Código do cliente no SICLI.
Perfil usuário Alçada: Código do usuário de alçada.
Pendência: Número da pendência.
Data Pendência: Data da pendência.
Situação Pendência: Situação da pendência.
Data Situação: Data da situação.
Usuário Gerador: Código do usuário gerador da pendência.
Hora Geração: Hora de geração da pendência.
Justificativa: Texto descrevendo o motivo da excepcionação.
Reiteração: Número da reiteração.
Data Reiteração: Data da reiteração.
Usuário Autorizador: Código do usuário autorizador da reiteração.
Perfil usuário Autorizador: Alçada do usuário autorizador.
Data Autorização: Data da autorização de reiteração.
Hora Autorização: Hora da autorização de reiteração.
Data Início Original: Data inicial de vigência cadastrada.
Data Fim Original: Data final de vigência cadastrada.
Data Fim (nova): Data final de vigência nova.
Resposta: Texto descrevendo o motivo da alteração.
Descrição: Item pendente.
Conteúdo Anterior: Valor padrão.
Conteúdo Negociado: Valor negociado.
Parecer: Situação do item.
