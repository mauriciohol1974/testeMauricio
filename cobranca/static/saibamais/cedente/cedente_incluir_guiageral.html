Incluir Cedente - Guia Geral

Descrição da Guia:
Nesta guia serão cadastrados os dados para identificação do cedente.
 
Botões de Ação:
Confirmar: Confirma a inclusão dos dados na guia geral.
Cancelar: Cancela a inclusão nesta guia e fecha-a.
 
Campos:
Tipo de Cobrança: Seleciona o tipo de cobrança.
Se for "Convencional", a 'Guia Cedente Eletrônico' ficará como "Não Permitido", não podendo ser editada.
Se for "Convencional", o campo 'Cedente Centralizador' será desabilitado.
Protesto Automático: Seleciona se protesto do título será automático ou não (D0).
Prazo Protesto: Prazo em dias para envio do título vencido ao cartório.
Prazo Devolução: Prazo em dias, após vencimento, para devolução do título ao cedente.
Extrato Mov. Títulos: Selecionar se tem extrato de movimentação de títulos ou não.
Se este campo for "não", o campo 'Destino Extrato Mov.' será desabilitado.
Destino Extrato Mov.: Seleciona modo de envio do extrato de movimetação de títulos.
Extrato Mov. Déb./Créd.: Selecionar se tem extrato de movimentação para débitos e créditos ou não.
Se este campo for "não", o campo 'Destino Extrato Déb./Créd.' será desabilitado.
Este campo só poderá ser "sim" se o campo 'Extrato Mov. Títulos' for "sim".
Destino Extrato Déb./Créd.: Seleciona modo de envio do extrato de movimetação para débitos/créditos.
Inventário Mês: Selecionar se tem inventário do mês ou não.
Tipo de Juros Dia: Seleciona o tipo de juros por dia cobrado sobre o valor do título após o vencimento.
Se o tipo de juros for "Juros Caixa" ou "Sem Juros", o campo 'Percentual Juros Dia' será desabilitado.
Percentual Juros Dia: Percentual de juros por dia cobrado sobre o valor do título após o vencimento.
Recebimento em Cheque: Se aceita recebimento em cheque ou não.
Percentual Multa: Percentual da multa cobrado sobre o valor do título após o vencimento.
Prazo para Multa: Prazo em dias para cobrança da multa.
Banco de Sacados: Se possui banco de sacados ou não.
Cliente Conta Garantida: Se o cliente vai Descontar Títulos ou não.
Cliente Externo: Se cliente externo ou não.
Campo desabilitado para Pessoa Física.
Cliente Giro Caixa: Se utiliza GiroCaixa ou não.
Campo desabilitado para Pessoa Física.
Se for "Convencional" ou "Eletrônica", o campo 'Setor' terá apenas a opção "Privado Pessoa Jurídica".
Setor: Setor de atuação da Pessoa Jurídica.
Campo desabilitado para Pessoa Física.
Porte: Porte da empresa da Pessoa Jurídica.
Campo desabilitado para Pessoa Física.
Natureza: Natureza do cliente Pessoa Jurídica.
Campo desabilitado para Pessoa Física.
Define as opções do campo 'Ramo de Atividade'.
Ramo de Atividade: Ramo de atividade da Pessoa Jurídica.
Campo desabilitado para Pessoa Física.
Exclusão Automática: Seleciona se exclusão automática do cedente da base, após dias sem movimentação.
Modalidade de Título: Selecionar modalidade de título.
Se for "Cobrança Caucionada", o campo 'Conta Caução' da 'Guia Contas Déb. Créd. e Rateio' é obrigatório.
PV de Vinculação: Código da Unidade PV de vinculação do cliente, onde foi contratada a cobrança.
Este código é automaticamente preenchido com a mesma Unidade PV do filtro.
Cedente Centralizador: Preenchido se o cedente for eletrônico.
Banco Correspondente: Código do banco correspondente que contratou a cobrança bancária CAIXA.
Cobrança Sem Boleto: Se cobrança sem boleto ou não.
Campo desabilitado para edição.
Situação do Cedente: Indicador da situação do cedente, se em Teste ou Produção.
