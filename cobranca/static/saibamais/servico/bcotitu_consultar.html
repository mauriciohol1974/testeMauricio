Banco de Títulos - Consultar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo consultar títulos do banco de títulos.
 
Botão de Ação:
Alterar: Encaminha para tela de alteração.
Voltar: Retorna para funcionalidade de ações no título.
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física do cedente.
Tipo de pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
Código Cliente (COCLI): Código do cliente no SICLI.
Nosso Número: Código identificador do título no banco.
Meio de Entrada: Canal utilizado como meio de entrada do título no sistema do banco.
Valor: Valor do título.
Situação: Situação do título.
 
Guia Dados Principais:
Campos:
Número Documento: Código identificador do título para o cedente.
Data de Vencimento: Data de vencimento do título.
Moeda: Tipo da moeda.
Aceite: Tipo de aceite.
Protesto: Se protesto de título ou não.
Prazo de Protesto/Devolução: Prazo e dias para protesto/devolução.
Endosso: Se precisa endosso ou não.
Espécie de Título: Espécie do título de cobrança.
Nome do Sacado: Nome do sacado do título.
Tipo Pessoa Sacado: Tipo de pessoa do sacado.
CPF/CNPJ do Sacado: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do sacado.
CEP do Sacado: Número do CEP do sacado.
UF do Sacado: Sigla da unidade federação do sacado.
Endereço do Sacado: Endereço do sacado.
Número do Sacado: Número do endereço do sacado.
Complemento do Sacado: Complemento do número do sacado.
Bairro do Sacado: Bairro do sacado.
Município do Sacado: Cidade do sacado.
e-mail do Sacado: E-Mail do sacado.
PV Cobrador: Código do PV Cobrador.
 
Guia Dados Complementares:
Campos:
Tipo Juros Mês: Tipo de juros por mês.
Percentual Juros Mês: Percentual de juros por mês.
Abatimento: Valor do abatimento no título.
Nome Sacador/Avalista: Nome do sacador/avalista.
Tipo Pessoa Sacador: Tipo de pessoa do sacador.
CPF/CNPJ Sacador: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do sacador.
Percentual: Percentual para Desconto/Multa.
Valor: Valor para Desconto/Multa.
Dias: Quantidades de dias para prazo limite de desconto ou prazo de multa.
Data: Data para prazo limite de desconto ou prazo de multa.
 
Guia Dados Complementares:
Campos:
Data Baixa/Liquidação: Data da baixa/liquidação do título.
Canal de Liquidação: Canal utilizado para liquidação do título.
Data de Pagamento: Data de pagamento do título.
Float: Quantidade de dias após a data do pagamento para crédito na conta do cedente.
Data de Crédito: Data para crédito na conta do cedente.
Valores Lançados: Valor total recebido para cada item.
 
Guia Dados Encargos e Abatimentos:
Campos:
Data de Emissão: Data de emissão do título.
Data de Entrada: Data de entrada do título no sistema.
Previsão Protesto/Devolução: Data de previsão para protesto/devolução do título.
Modalidade Título(Carteira): Código que identifica a modalidade (carteira) do título.
Abatimento: Valor de abatimento no título.
Custas Cartorárias: Valor das custas com o cartório pelo encaminhamento do título a protesto.
Prazo: Prazo de dias para o abatimento ou encargo.
Data: Data limite para abatimento ou encargo.
Percentual: Percentual para o abatimento ou encargo.
Valor: Valor do abatimento ou encargo.
