Manter Banco de Sacados - Alterar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo alterar o banco de sacados.
 
Botões de Ação:
Confirmar: Confirma a alteração do banco de sacados.
Voltar: Retorna para funcionalidade listar banco de sacados.
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome fantasia para Pessoa Jurídica ou nome para Pessoa Física.
Tipo de Pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica).
Código Cliente (COCLI): Código do cliente no SICLI.
Banco de Sacados: Número do banco de sacados.
Nome: Nome do banco de sacados a ser alterado.
