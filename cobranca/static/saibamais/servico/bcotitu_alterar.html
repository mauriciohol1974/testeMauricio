Banco de Títulos - Alterar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo alterar o título selecionado do banco de títulos.
 
Botões de Ação:
Confirmar: Confirma a alteração do título e emite um novo título para impressão.
Voltar: Retorna para funcionalidade de ações no título.
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física do cedente.
Tipo de Pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
Código Cliente (COCLI): Código do cliente no SICLI.
Nosso Número: Código identificador do título no banco.
Meio de Entrada: Canal utilizado como meio de entrada do título no sistema do banco.
Valor: Valor do título.
Situação: Situação do título.
Comunicação ao Sacado: Seleciona se comunica alteração ao sacado ou não (emissão e postagem)
Cobrar Tarifa: Seleciona se haverá cobrança de tarifa pela alteração ou não.
 
Guia Dados Principais:
Botão de Ação:
Procurar: Pesquisa de CEP.
 
Campos:
Número Documento: Código que identifica o título para o cedente.
Data de Vencimento: Data de vencimento do título.
Campo desabilitado para edição.
Moeda: Tipo da moeda.
Campo desabilitado para edição.
Aceite: Seleciona tipo de aceite.
Protesto: Se protesto de título ou não.
Campo desabiltado para edição.
Prazo de Protesto/Devolução: Prazo de dias para protesto ou devolução.
Endosso: Seleciona se precisa endosso ou não.
Espécie de Título: Seleciona a espécie do título de cobrança.
Nome do Sacado: Nome do sacado do título.
Tipo Pessoa Sacado: Seleciona tipo de pessoa do sacado.
CPF/CNPJ do Sacado: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do sacado.
CEP do Sacado: Número do CEP do sacado.
UF do Sacado: Seleciona sigla da unidade federação do sacado.
Endereço do Sacado: Endereço do sacado.
Número do Sacado: Número do endereço do sacado.
Complemento do Sacado: Compelemento do número do sacado.
Bairro do Sacado: Bairro do sacado.
Município do Sacado: Cidade do sacado.
e-mail do Sacado: E-Mail do sacado.
PV Cobrador: Código do PV Cobrador.
Campo desabilitado para edição.
 
Guia Dados Complementares:
Campos:
Tipo Juros Mês: Seleciona tipo de juros por mês.
Se for "Percentual de Juros Cedente", o campo 'Percentual Juros Mês' será habilitado.
Percentual Juros Mês: Percentual de juros por mês, a ser aplicado ao título, se pago após o vencimento.
Abatimento: Valor de abatimento do título.
Nome Sacador/Avalista: Nome do sacador/avalista.
Tipo Pessoa Sacador: Tipo de pessoa do sacador.
CPF/CNPJ Sacador: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do sacador.
Percentual: Percentual para Desconto/Multa.
Se este campo for preenchido, o campo 'Valor' será desabilitado.
Valor: Valor para Desconto/Multa.
Se este campo for preenchido, o campo 'Percentual' será desabilitado.
Dias: Quantidades de dias para prazo limite de desconto ou prazo de multa.
Se este campo for preenchido, o campo 'Data' será desabilitado.
Data: Data para prazo limite de desconto ou prazo de multa.
Se este campo for preenchido, o campo 'Dias' será desabilitado.
