Solicitar 2a. via de Extrato - Consultar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo consultar solicitação da 2ª via de extrato.
 
Botões de Ação:
Alterar: Encaminha para a funcionalidade de alteração de solicitação da 2ª via de extrato.
Excluir: Exclui a solicitação da 2ª via de extrato.
Voltar: Retorna para tela de filtrar.
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física do cedente.
Tipo de pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
Código Cliente (COCLI): Código do cliente no SICLI.
Tipo de Extrato: Tipo de extrato selecionado.
Data início: Data de início do extrato que está sendo solicitado.
Data fim: Data final do extrato que está sendo solicitado.
Data solicitação: Data de inclusão da solicitação de 2ª via do extrato.
Envio do Boleto: Modo de entrega da 2ª via de extrato.
Usuário: Código do Usuário de alçada resposnável pela inclusão da solicitação de 2º via de extrato.
