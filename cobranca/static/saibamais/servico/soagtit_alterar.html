Manter Solic./Agend. de Emissão de Título - Alterar

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo alterar solicitação ou agendamento de emissão de título.
 
Botões de Ação:
Confirmar: Confirma a alteração da solicitação ou agendamento de emissão de título.
Voltar: Retorna para funcionalidade de filtrar solicitação ou agendamento de emissão de título para manutenção.
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física.
Tipo de Pessoa: Tipo de pessoa do cedente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cedente.
Código Cliente (COCLI): Código do cliente no SICLI.
Banco de Sacados: Número do banco de sacados.
Nome Banco de Sacados: Nome do banco de sacados.
Tipo de Requisição: Tipo de requisição do título.
Registro: Registro do título ou não no sistema.
Campo desabilitado para edição.
Espécie: Seleciona espécie do título de cobrança.
Forma Obtenção Valor Título: Seleciona o meio de obtenção do valor para o título.
Se for "Sacado", os campos 'Valor' serão desabilitados.
Valor: Valor do título.
Forma Obtenção Envio Boleto: Seleciona o meio de obtenção de envio do Boleto. Se for "Sacado", o campo 'Envio do Boleto' será desabilitado.
Envio do Boleto: Modo de envio do Boleto.
Endosso: Seleciona se necessita endosso ou não.
Aceite: Seleciona o tipo de aceite.
Protesto Automático: Seleciona se é para protesto automático ou não.
Prazo Protesto/Devolução: Prazo de dias para protesto e devolução do título.
Moeda: Tipo de moeda do título.
Campo desabilitado para edição.
Tipo Juros Dia: Seleciona o tipo de juros por dia que incidirão sobre o título liquidado após o vencimento.
Se for "Juros Cedente", o campo 'Percentual Juros Dia' será habilitado para edição.
Percentual Juros Mês: Percentual de juros por mês.
Percentual - Desconto/Multa: Percentual para desconto ou multa do título.
Se este campo for preenchido, o campo 'Valor' será desabilitado.
Valor - Desconto/Multa: Valor para desconto ou multa do título.
Se este campo for preenchido, o campo 'Percentual' será desabilitado.
Dias - Prazo Limite/Prazo Multa: Quantidade de dias para prazo limite do desconto ou multa.
Se este campo for preenchido, o campo 'Data' será desabilitado.
Data - Prazo Limite/Prazo Multa: Data para prazo limite de desconto ou multa.
Se este campo for preenchido, o campo 'Dias' será desabilitado.
Este campo não existe para agendamento.
 
Agendamento:
Dia de Vencimento: Dia agendado para vencimento do título.
Forma de Vencimento: Forma para vencimento do título.
Dia de Emissão Boleto: Dia agendado para emissão do Boleto.
Forma de Emissão Boleto: Forma para emissão do Boleto.
Agendamento Permanente: Se agendamento é permanente ou não.
