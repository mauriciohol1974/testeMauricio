Manter Sacado - Filtro

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo filtrar o cedente e banco de sacados para permitir a manutenção de sacados.
 
Botões de Ação:
Ok: Efetua filtragem de acordo com os parâmetros informados.
 
Campos:
Cedente: Código do cedente.
Ao preencher, o campo 'Número Banco de Sacados' será habilitado para edição.
Número Banco de Sacados: Código do banco de sacados. Se não preenchido retorna todos os bancos de sacados vinculados ao cedente informado.
Ao preencher este, o campo 'CPF/CNPJ Sacado' será habilitado para edição.
CPF/CNPJ Sacado: Seleciona o tipo de pessoa e ao lado o número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica). Se não preenchido obtém todos os bancos de sacado do sistema.
Ao preencher este, o campo 'Código Sacado' será habilitado para edição.
Código Sacado: Código do sacado no Banco de Sacados. Se não preenchido retorna todos os sacados vinculados ao banco de sacados informado.
