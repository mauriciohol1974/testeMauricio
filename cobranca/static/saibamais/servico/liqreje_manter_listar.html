Liquidações Rejeitadas - Listar

Descrição Funcionalidade:
Esta funcionalidade é listar para manutenção de liquidações rejeitadas.
 
Botões de Ação:
Executar Ação: Executa a funcionalidade selecionada.
Cancelar Ação: Cancela a funcionalidade selecionada.
Consultar: Encaminha para funcionalidade de Consulta da liquidação rejeitada selecionada.
Voltar: Retorna para funcionalidade de filtragem das liquidações rejeitadas.
 
Campos:
Opção: Tipo da pedência ou consulta selecionada.
PV Recebedor/Centralizador: Código do PV Recebedor/Centralizador.
Meio Liquidação: Canal utilizado como meio de liquidação do título.
Data (a partir de): Data de início da filtragem.
Este campo virá em branco, caso não tenha sido preenchido no filtro.
Data Ref.: Data da liquidação rejeitada.
NSeq: Código identificador do numero sequencial do registro rejeitado no processamento da liquidação do dia.
Cedente: Código do Cedente.
Nosso Número: Código identificador do título no banco.
Valor Recebido: Valor recebido do título liquidado.
Erro: Código do erro que motivou a rejeição do título.
Passe o mouse em cima do código para descrição do Erro.
Seq: Código identificador do numero sequencial dos titulos liquidados via compensação.
Tipo de Carteira: Código identificador do tipo da carteira do título rejeitado.
