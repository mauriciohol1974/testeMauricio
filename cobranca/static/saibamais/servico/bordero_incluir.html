Incluir Borderô

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo incluir um novo borderô de títulos.
 
Botões de Ação:
Confirmar: Confirma a inclusão do novo borderô.
Voltar: Retorna para funcionalidade de filtrar borderô para inclusão.
 
Campos:
Cedente: Código do cedente.
Nome Cedente: Nome da Pessoa Jurídica ou da Pessoa Física do cedente.
Tipo de Pessoa: Tipo de pessoa do cliente.
CPF/CNPJ: Número do CPF(Pessoa Física) ou CNPJ(Pessoa Jurídica) do cliente.
Código Cliente (COCLI): Código do cliente no SICLI.
 
Guia Borderô:
Campos:
Somente para Protesto: Seleciona se título do borderô é somente para protesto ou não, nesse caso, o sistema não emitirá o Boleto.
Se for "sim" os campos 'Modalidade Título', 'Protesto Automático', 'Prazo Protesto/Devolução', 'Emissão Boleto' e 'Envio do Boleto' serão desabilitados.
Modalidade Título: Seleciona modalidade do título.
Data do Movimento: Data em que a solicitação está sendo feita.
PV de Vinculação: Código do PV de vinculação do cedente.
Quantidade: Quantidade de títulos para o borderô.
Valor: Valor dos títulos para o borderô.
Endosso: Seleciona se endosso ou não.
Espécie de Título: Seleciona espécie do título de cobrança.
Aceite: Seleciona o tipo de aceite.
Moeda: Tipo de moeda.
Campo desabilitado para edição.
Protesto Automático: Seleciona se protesto automático ou não.
Prazo de Protesto/Devolução: Quantidade de dias para protesto ou devolução do título.
Emissão Boleto: Seleciona quem emitirá o Boleto.
Se for "Cedente", o campo 'Envio do Boleto' será desabilitado.
Envio do Boleto: Seleciona o modo de envio do Boleto ao sacado.
 
Guia Dados Complementares:
Botão de Ação:
Editar: Abre uma janela para edição do campo.
 
Campos:
Tipo Juros Dia: Seleciona o tipo de juros a ser aplicado ao título se o pagamento for após o vencimento.
Percentual Juros Mês: Percentual de juros por mês.
Percentual - Desconto/Multa: Percentual para desconto ou multa do título do borderô.
Dias - Prazo Limite/Prazo Multa: Quantidade de dias para prazo limite do desconto ou para prazo multa.
Mensagem Ficha de Compensação: Mensagem para ser impressa na ficha de compensação.
