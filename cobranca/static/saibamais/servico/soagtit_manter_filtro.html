Manter Solic./Agend. de Emissão de Título - Filtro

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo filtrar para permitir a solicitação ou agendamento de emissão de títulos para manutenção.
 
Botões de Ação:
Ok: Encaminha para funcionalidade de listagem da solicitação ou agendamento de emissão de títulos para manutenção.
 
Campos:
Cedente: Código do cedente.
Número Banco de Sacados: Código do banco de sacados.
Se não preenchido retorna todos os bancos de sacados vinculados ao cedente informado.
Tipo de Requisição: Seleciona o tipo de requisição.
