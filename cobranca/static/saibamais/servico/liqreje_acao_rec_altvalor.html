Liquidações Rejeitadas - Recomando Especial

Descrição Funcionalidade:
Esta funcionalidade é acionar recomando especial de uma liquidação rejeitada.
 
Botões de Ação:
Confirmar: Confirma a inclusão das alterações na liquidação rejeitada.
Voltar: Retorna para funcionalidade de listagem da liquidação rejeitada.
 
Campos:
Opção: Tipo da pedência ou consulta.
PV Recebedor/Centralizador: Código do PV Recebedor/Centralizador.
Meio Liquidação: Meio de liquidação.
Data (a partir de): Data de início da filtragem.
Este campo virá em branco, caso este não tenha sido preenchido no filtro.
Data de Referência: Data da liquidação rejeitada.
Sequência: Código identificador do número sequencial do registro rejeitado no processamento da liquidação do dia.
Cedente: Código do cedente.
Nosso Número: Código identificador do título no banco.
Valor Recebido: Valor recebido do título liquidado.
Data Situação Anterior: Data da última situação rejeitada.
Código do Erro: Código identificador do Erro.
Erro: Descrição do Erro.
Número do Lote: Código identificador do número de lote quando ocorrer liquidação do título via Compensação.
Sequência do Lote: Código identificador do número sequencial dos títulos liquidados via compensação.
Data de Movimento: Data da última movimentação.
Cedente Corrigido: Código do novo cedente.
Nosso Número Corrigido: Código identificador novo do título no banco.
Lote Corrigido: Código identificador novo do número de lote.
Valor Corrigido: Valor corrigido do título liquidado.
