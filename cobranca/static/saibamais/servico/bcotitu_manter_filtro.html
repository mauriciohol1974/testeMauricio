Banco de Títulos - Filtro

Descrição Funcionalidade:
Esta funcionalidade tem como objetivo filtrar o cedente para manutenção no banco de títulos.
 
Botões de Ação:
Ok: Encaminha para funcionalidade de manutenção do banco de títulos.
 
Campos:
Cedente: Código do cedente.
Nosso Número: Código identificador do título no banco.
Se este for selecionado, os campos 'Situação do Título' e 'Classificado por' serão desabilitados.
Situação do título: Seleciona a situação do título.
Se este for selecionado, o campo 'Nosso Número' será desabilitado.
Classificado por: Seleciona a ordem de classificação.
Este campo só será habilitado, se houver a seleção do campo 'Situação do Título'.
